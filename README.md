# BudgetManagerService

BudgetManagerService is the backend service for the *BudgetManager* application.  
It is implemented in **Java with Spring Boot** and provides REST APIs used by the desktop client built with JavaFX.

This project was developed for **academic purposes**, following the specifications of the course professor.
